Palindrome

Palindromes are strings that form the same word if it is reversed. Eg: madam <-> madam, level -> level

1. Using reverse() - First convert string to array and apply reverse() to it and later compare input string with reversed string. 


                const palindrome = (str) => {
                    return str === str.split('').reverse().join('') ? true : false
                }
                
                const test1 = palindrome('level');
                console.log(test1); // true
                const test2 = palindrome('god');
                console.log(test2); //false

2. with every() - The every() method tests whether all elements in the array pass the test implemented by the provided function. It returns a Boolean value. 

                const palindrome1 = (str) => {
                  return str.split('').every((char, i) => {
                      return char === str[str.length - i - 1]
                  })
              }
              
              const test4 = palindrome1('level');
