Reverse an Integer

Reverse an integer

Eg: reverseNumber(123); // 321 reverseNumber(-123); // -321 reverseNumber(200); //2 


            const reverseNumber = (myNumber) => {
                return parseInt(myNumber.toString().split('').reverse().join('')) * Math.sign(myNumber)
            }
            
            console.log(reverseNumber(123)); // 321
            console.log(reverseNumber(-123)); //-321
            console.log(reverseNumber(200)); //2
